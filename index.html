eMed-DNA Tutorial
Input Files Description

The user will need to provide this three files as input.

The Homo_sapiens genome folder contains DNA sequence of 24 chromosomes of human body in FASTA format. 'X' and 'Y' chromosomes are marked as 23rd and 24th chromosome respectively. The lengths of all chromosomes are in chrome_length.txt file. Chromosome number, start and end positions of genes in genome sequence are in gene_list.txt file.

Entering new patient details

The user can select the genome folder of the patient and the gene list and length of chromosomes. Then he can operate the patient's data stored inside genome.

The page will be updated by showing the data stored inside patient's genome.

Viewing files

The user can view any file stored inside genome

The page will be updated by viewing the file in desktop's default viewer for that filetype.

Inserting DICOM Files

The user can insert any dicom file from his desktop in the genome sequence of the patient.

The page will be updated by inserting the selected dicom files in the genome and viewing them in the listbox.

Inserting non-DICOM Files

The user can insert any non-DICOM file from his desktop in the genome sequence of the patient.

The page will be updated by inserting the selected non-DICOM files in the genome and viewing them in the listbox.

Deleting files from genome

The user can select multiple files (both DICOM and non-DICOM) and delete them from genome sequence of the patient.

The page will be updated by deleting the selected files in the genome and removing them from the listbox.

Saving files from genome sequence.

The user can save files one at a time from patient's genome sequence to desktop.

The page will be updated by saving the selected files from the genome in desktop and showing them in the desired location.
